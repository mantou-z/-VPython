# VPython 实现单摆运动仿真
## 安装VPython
cmd控制台中运行：
```
pip install vpython
```
## 运行 **单摆.py**
cmd控制台中运行：
```
python 单摆.py
```
## 使用说明
提示  give a force?(y or n)  
如果观察正常摆动，输入 n 回车，之后会跳出浏览器页面进行动画展示  
如果观察施加外力，输入 y 回车，之后会提示输入 KP KD 两个数值，输入数值后回车，进入动画演示，浏览器动画页面单击鼠标会切换 施加/取消外力 两种状态。  
其他各项数值可在程序中直接修改。
